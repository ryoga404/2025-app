アプリ名：（仮称）家計簿アプリ

アプリ概要：
買い物のレシートと食品のバーコードをスキャンし、記録していく。
収入・支出を可視化する。
賞味期限（消費期限）をJANコードから読み取り取得


機能：
・OCR（レシート画像の文字列化）
・ユーザ登録
・収入支出グラフ化
・過去の記録との比較
・賞味期限通知


その他：
・ユーザ管理はDBで。
・ユーザIDとパスワード（DB登録時はソルトを生成し保存。パスは保存なし）

--------------------
［会員登録］
１、（ユーザ）ユーザIDとパスワードを登録
２、（アプリ）ランダムソルトを発生させる
３、（アプリ）パスワード＋ソルトをSHA-512でハッシュ化
４、（アプリ）DBにユーザIDとソルトとSHA-512ハッシュ値を保存

［ログイン］
１、（ユーザ）ユーザIDとパスワードを入力
２、（アプリ）パスワードをDBソルトと合わせ、SHA-512でハッシュ化
３、（アプリ）DBハッシュ値と照合
４、（アプリ）一致すればログイン
--------------------

使用想定環境：

［プロトタイプ１］DB内に家計簿データ保存・表示・削除

datas TABLE
id int auto_increment primary key
date DATE not null
type VARCHAR(10) not null check(type in ('収入','支出'))
subtype VARCHAR(20)
ammount int not null check(ammount >= 0)

・抽出
typeでGROUP BYで合計→Dartに返す
日付の範囲などで合計など出力
支出のうち〇〇に最も金額をかけている
